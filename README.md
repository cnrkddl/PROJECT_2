# Contextual Video Ad Insertion System (맥락 기반 영상 광고 삽입 시스템)

## 📌 프로젝트 개요
이 프로젝트는 단순히 영상 중간에 임의로 광고를 배치하는 것을 넘어, **AI가 영상의 흐름과 시각적 요소를 인간처럼 이해하고 가장 자연스러운 광고 경험을 설계**하는 것을 목표로 합니다. 시청자의 시청 경험을 해치지 않으면서도 광고 효율을 극대화하는 지능형 시스템입니다.

### 핵심 목적
1. **시청 경험 최적화**: 영상의 몰입을 방해하는 무분별한 중간 광고 대신, 씬(Scene)의 전환이나 무음 구간 등 영상 흐름이 끊기지 않는 최적의 타이밍(Golden Time)을 AI가 자동으로 감지합니다.
2. **광고 효율 극대화**: 영상 속에 등장하는 객체(Object), 장소(Place), 인물의 대사(Script)를 분석하여 해당 장면의 맥락(Context)과 가장 잘 어울리는 상품을 추천합니다.

---

## ⚙️ 주요 시스템 워크플로우 (Data Pipeline)
영상 업로드부터 분석 결과 반환까지의 파이프라인은 다음과 같이 구성됩니다.

1. **전처리 (Preprocessing)**
   - 업로드된 VOD를 샷(Shot) 및 씬(Scene) 단위로 분할합니다.
   - 비디오와 오디오를 분리하여 독립적으로 분석할 수 있는 환경을 구축합니다.

2. **멀티모달 분석 (Multimodal Analysis)**
   - **Vision**: 화면 속 상품, 장소, 분위기, 객체를 인식합니다.
   - **Audio/NLP**: Whisper 모델을 통해 대사를 텍스트로 변환(ASR)하고, NLP 기법을 적용해 의미를 벡터화하여 전체적인 맥락을 파악합니다.

3. **삽입점 감지 (Insertion Detection)**
   - 씬 전환점, 오디오 무음 구간, 감정의 변화 지표 등을 종합적으로 스코어링합니다.
   - 광고가 들어가기에 가장 적절한 '골든 타임'을 계산합니다.

4. **데이터 저장 및 제공**
   - 분석 결과물(메타데이터, 삽입 포인트 프레임 정보 등)을 JSON 형태로 생성합니다.
   - 관계형 데이터베이스(PostgreSQL)에 구조화하여 저장합니다.
   - 데스크탑 전용 앱 또는 RESTful API 형태로 메타데이터를 클라이언트에게 제공합니다.

---

## 🚀 기술적 차별점 (Core Strengths)

- **비동기 병렬 처리 아키텍처**: 
  Redis 메시지 큐와 Docker 컨테이너 오케스트레이션을 활용하여, 무거운 대용량 VOD 분석 작업을 중단이나 병목 현상 없이 안정적으로 처리합니다.
- **맥락 지능 (Context Intelligence)**: 
  단순한 키워드 매칭이나 규칙 기반 동작을 넘어섭니다. 영상의 시각 정보와 음성 정보를 결합한 멀티모달 딥러닝(Multimodal Deep Learning)을 활용하여 광고가 해당 지점에 노출되어야만 하는 '명분'을 부여합니다.
